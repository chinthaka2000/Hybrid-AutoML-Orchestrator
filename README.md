# Hybrid AutoML Orchestrator

A Hybrid AutoML Orchestrator that combines Policy-Driven AutoML with Cognitive Reasoning (LLM).

## Structure

- **mcp**: Policy-driven logic (Model Control Platform)
- **automl**: AutoML components (H2O.ai)
- **orchestrator**: Pipeline orchestration
- **monitoring**: Data drift and performance monitoring
- **registry**: Model registry and metadata
- **deployment**: Inference and rollback
